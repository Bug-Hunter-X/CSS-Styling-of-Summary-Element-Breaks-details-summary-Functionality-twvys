This is an example of an uncommon HTML bug. It's related to using the <details> and <summary> elements. The issue occurs when you try to style the <summary> element with CSS to make it look like a button, but clicking it doesn't open or close the details section as expected.

Here's the buggy code:

<details>
  <summary style="cursor:pointer; border:1px solid #ccc; padding:5px; background-color:#eee;"><span>Click Me</span></summary>
  <p>This is the hidden content.</p>
</details>

The problem is subtle. While the styling works visually, the click event might not properly trigger the opening/closing of the details element. This can be inconsistent across different browsers.
